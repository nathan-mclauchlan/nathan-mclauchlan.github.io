Id And Classes

ID

An id attribute is used to point towards an HTML element like h1, when you give an id attribute to an element it is unique to that speicfic element meaning that you can only use it once, whenever you give your chosen element the id attribute it will look for styles you have selected from your CSS file and will be applyed across every HTML document that is linked to your main CSS file.
